# CPRG-306 Week 4

# Interactivity

---

## Agenda

- Learn enough JavaScript to make our web pages interactive.
- Create a counter component.
- Create a managed form component.
- Style our components with Tailwind CSS.

---

## JavaScript

- Higher order functions
- Arrow functions

---

## Counter component

- Create a counter component
- Use the useState hook to manage state
- Add conditional rendering to the component
- Add Tailwind states to the component

---

## Managed form component

- Create a form to add a dog
- Use the useState hook to manage state of each form control
- Use the onChange event handler to update state
- Upon submission, create a dog object and display it in the console
